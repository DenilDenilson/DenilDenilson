## Proyectos que quiero mostrar mejor

### `rat-cli`

Una herramienta CLI en C para recorrer proyectos y preparar contexto útil para LLMs.

La idea nació de una necesidad simple: dejar de copiar carpetas enteras y empezar a pasarle a la IA solo lo que realmente importa.

**Temas:** C, CLI tools, archivos, contexto para LLMs, tooling para desarrolladores.

---

### `demi-software`

Backend experimental para un asistente personal basado en voz.

La idea es conectar audio, transcripción, interpretación de intención, memoria y acciones automatizadas.

**Temas:** FastAPI, IA aplicada, audio, automatización, workflows, backend.

---

### Proyectos IoT / monitoreo

He trabajado en sistemas que leen datos de sensores, los envían por MQTT y los muestran en plataformas web.

**Temas:** backend, sensores, MQTT, Modbus, ESP32, dashboards, datos en tiempo real.
