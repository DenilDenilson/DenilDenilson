## Sobre mí

Soy desarrollador con enfoque en **backend, automatización, sistemas e IoT**.

Me interesa construir software que no se quede solo en la interfaz: APIs, servidores Linux, bases de datos, herramientas internas, automatizaciones con IA, sistemas de monitoreo y proyectos que interactúan con sensores, hardware o procesos reales.

Mi formación en Ingeniería Física me llevó a moverme entre software, electrónica, datos y sistemas. Por eso disfruto especialmente los proyectos donde hay que entender el problema, diseñar una solución práctica y dejar algo funcionando, documentado y desplegable.

---

## Qué estoy construyendo

Actualmente estoy usando este GitHub como una vitrina técnica de proyectos propios, experimentos y herramientas que voy documentando.

Me enfoco principalmente en:

* **Backend y APIs** con Python, FastAPI, Node.js y TypeScript.
* **Automatización de procesos** con scripting, workflows e IA aplicada.
* **Software conectado a hardware** usando MQTT, Modbus, ESP32, Arduino y sensores.
* **Herramientas para desarrolladores** orientadas a CLI, contexto para LLMs y productividad.
* **Productos web propios** con Astro, React, Next.js y despliegues en servidores o plataformas cloud.
